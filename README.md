# 🚀 Doneo Box — AI Directory

> Um diretório inteligente e ultra-rápido para explorar ferramentas de Inteligência Artificial em um só lugar.

O **Doneo Box** é um motor de busca e catálogo centralizado de ferramentas de IA, pensado para ser **leve, escalável e extremamente intuitivo**.
A proposta é simples: eliminar o caos e organizar o ecossistema de IA com eficiência.

## ⚡ Destaques

* 🔍 **Busca Semântica Inteligente**
  Sistema de scoring com pesos que correlaciona termos com **nomes, descrições e tags** para resultados mais relevantes.

* 🧩 **Categorização Dinâmica**
  Filtros reativos com mais de **30+ categorias** (Texto, Imagem, Código, Vídeo e mais).

* 🧠 **Indexação Robusta**
  +650 ferramentas cadastradas com metadados detalhados (**Grátis, Freemium, Pago**).

* 🧱 **Arquitetura Modular**
  Código dividido em módulos independentes → mais organização, manutenção fácil e escalabilidade.

* 🎨 **Interface Adaptativa**
  Design minimalista com suporte nativo a **Dark/Light Mode** e responsividade total.
